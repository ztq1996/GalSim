Detailed Description

A class representing an Angle.

Angles are a value with an AngleUnit.

You typically create an Angle by multiplying a number by an AngleUnit. e.g.

Angle pixel = 0.27 * arcsec;
Angle ra = 13.4 * hours;
Angle dec = -32 * degrees;
Angle theta = PI/2 * radians;

You can also use the constructor explicitly, which takes a value and a unit:

Angle theta(90, degrees);

Since extracting the value in radians is extremely common, we have an accessor to do this quickly:

x = theta.rad();

It is equivalent to the more verbose:

x = theta / radians;

but without actually requiring the FLOP of dividing by 1.

Arithmetic with Angles include the following: (In the list below, x is a double, unit is an AngleUnit, and theta is an Angle.)

theta = x * unit
x = theta / unit
theta3 = theta1 + theta2
theta3 = theta1 - theta2
theta2 = theta1 * x
theta2 = x * theta1
theta2 = theta1 / x
theta2 += theta1
theta2 -= theta1
theta *= x
theta /= x

I/O:

os << theta

just outputs the value in radians.
