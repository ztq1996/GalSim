Function to simulate images (no added noise) from real galaxy training data.

This function takes a RealGalaxy from some training set, and manipulates it as needed to simulate a (no-noise-added) image from some lower-resolution telescope. It thus requires a target PSF (which could be an image, or one of our base classes) that represents all PSF components including the pixel response, and a target pixel scale.

The default rotation option is to impose a random rotation to make irrelevant any real shears in the galaxy training data (optionally, the RNG can be supplied). This default can be turned off by setting rand_rotate = False or by requesting a specific rotation angle using the rotation_angle keyword, in which case rand_rotate is ignored.

Optionally, the user can specify a shear (default 0). Finally, they can specify a flux normalization for the final image, default 1000.

Parameters:
real_galaxyThe RealGalaxy object to use, not modified in generating the simulated image.
target_PSFThe target PSF, either one of our base classes or an ImageView/Image.
target_pixel_scaleThe pixel scale for the final image, in arcsec.
g1First component of shear to impose (components defined with respect to pixel coordinates), default g1 = 0.
g2Second component of shear to impose, default g2 = 0.
rotation_angleAngle by which to rotate the galaxy (must be a galsim.Angle instance).
rand_rotateIf rand_rotate = True (default) then impose a random rotation on the training galaxy; this is ignored if rotation_angle is set.
rngA random number generator to use for selection of the random rotation angle. (optional, may be any kind of galsim.BaseDeviate or None)
target_fluxThe target flux in the output galaxy image, default target_flux = 1000.
imageAs with the GSObject.draw() function, if an image is provided, then it will be used and returned. If image=None, then an appropriately sized image will be created.
Returns:
A simulated galaxy image. The input RealGalaxy is unmodified.
