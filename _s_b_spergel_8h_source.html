32  // Constrain range of allowed Spergel index nu. Spergel (2010) Table 1 lists values of nu

34  // handle, however, so I'm setting the lower range to -0.85 instead. The upper limit is
35  // set by the boost::math::cyl_bessel_k function, which I found runs into overflow errors
