Construct a new Gaussian-distributed RNG, using the provided value as seed.

If lseed == 0, this means to use the time of day as the seed. Note that microsecond counter is the seed, so BaseDeviates constructed in rapid succession may not be independent.

Clear the internal cache.

void galsim::BaseDeviate::reset ( long  lseed)
inherited

Like seed(lseed), but severs the relationship between other Deviates.

Other Deviates that had been using the same RNG will be unaffected, while this Deviate will obtain a fresh RNG seeding by the current time.

void galsim::BaseDeviate::reset ( const BaseDeviate &  dev)
inherited

Make this object share its random number generator with another Deviate.

It discards whatever rng it had been using and starts sharing the one held by dev.

virtual void galsim::BaseDeviate::seed ( long  lseed)
virtualinherited

Re-seed the PRNG using specified seed.

If lseed == 0, this means to use the time of day as the seed. Note that microsecond counter is the seed, so BaseDeviates constructed in rapid succession may not be independent.

Parameters
[in]lseedA long-integer seed for the RNG.

Note that this will reseed all Deviates currently sharing the RNG with this one.
