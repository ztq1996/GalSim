  rel_err = 1.e-6,

Integrate a 1-dimensional function from min to max.

Example usage: 

>>> def func(x): return x**2
>>> galsim.integ.int1d(func, 0, 1)
0.33333333333333337
>>> galsim.integ.int1d(func, 0, 2)
2.666666666666667
>>> galsim.integ.int1d(func, -1, 1)
0.66666666666666674
Parameters:
funcThe function to be integrated. y = func(x) should be valid.
minThe lower end of the integration bounds (anything < -1.e10 is treated as negative infinity).
maxThe upper end of the integration bounds (anything > 1.e10 is treated as positive infinity).
rel_errThe desired relative error (default rel_err = 1.e-6)
abs_errThe desired absolute error (default abs_err = 1.e-12)
